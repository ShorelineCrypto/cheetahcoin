NENG Android 64bit Wallet Upgrade Guide without Putty
    1. Open Userland App and select Debian
    2. Log in Debian
    3. cd  Android_Userland_App/debian/arm64/
    4. wget https://github.com/ShorelineCrypto/nengcoin/releases/download/v2.1.0/nengcoin_2.1.0_arm64_linux-gnu.tgz
    5. tar xvfz nengcoin_2.1.0_arm64_linux-gnu.tgz
    6. rm -rf nengcoin_v1.9.1.2_u16_arm64 (remove previous nengcoin qt folder)
    7. rm nengcoin_v1.9.1.2_u16_arm64.tgz (remove previous nengcoin tgz file if exist)
    8. rm nengcoin_2.1.0_arm64_linux-gnu.tgz (remove current nengcoin tgz file)
